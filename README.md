instructions:- download the repository and extract the zip
1. backend
step1. open command prompt or any terminal of your choice
step2. navigate to backend  folder and then run "npm install"
step3. after installation of packages completes run : "node server.js" <--- this will run the vite app at localhost:8080 port

2. frontend
step1. open new window of same terminal and navigate to frontend folder
step2. run "npm install"
step3. run "npm run dev" <--- this will run the vite app at localhost:5173 port
